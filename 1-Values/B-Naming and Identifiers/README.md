# Naming and Identifiers

## What You Will Build
- A program to count animals.
- A program to determine if your friend's concert will be profitable or result in a loss.
- A program to decide how you can use your time to have a better morning.

## What You Will Learn
- Why writing code to solve a problem is preferable to writing out solutions and figuring them out on your own.

## Key Vocabulary
- ``Assignment``: The process of giving a value to a variable or constant.
- ``Constant``: A value that does not change after its assignment.
- ``Declaration``: The introduction of a variable or constant with its type.
- ``Identifier``: A name given to a variable, constant, function, etc., to uniquely identify it.
- ``Keyword``: A reserved word that has a special meaning in the programming language.
